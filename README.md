Employee Payroll Management System

A desktop payroll management application built with Python that helps organizations manage employee records, calculate salaries, and generate payroll reports through an easy-to-use graphical user interface.

Features

Add, update, and delete employee records

Automated salary calculation

Store payroll history

Generate payroll summaries

User-friendly graphical interface

Persistent data storage

Technologies Used
Python

Tkinter (GUI)

SQLite database
